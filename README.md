# Quick Search for Obsidian

This is a minimalist search plugin for Obsidian.

About this project:
Please note that I am not a professional developer. This plugin was created in collaboration with Gemini (AI). My role was to provide the functional requirements, design instructions, and troubleshooting feedback, while the AI generated the code structure.

I am hosting this code here to share this tool with the community. Feel free to explore, modify, and improve it—contributions are more than welcome!

Key Features:

- **Minimalist & Native**: Perfectly matches Obsidian’s theme.
- **Content Search**: Searches both file names and the actual content of your notes
- **Smart Features**: Includes custom folder shortcuts, dynamic icons for attachments, subheading navigation (#), and smart property previews
- **Configurable**: Easily customize colors and define your own folder filters directly in the settings.
